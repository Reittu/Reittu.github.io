Experience

Summer Project 2018 - multi-platform game (C#, Unity)

We, a team of three people, were tasked to build a multi-platform game meant to collect well-being data from children.

Due to the nature of the project, we had to solve a major problem conserning the obvious legal issues. We ended up arranging a solution, where the stored data itself is anonymous but the supervisors could identify players by session start and end times.

I did most of the programming while my team members worked on general design, level design, environment, characters and animations. The project also included a lot of planning together as well as meetings and demo sessions with the client.

I learned a lot about topics such as particle effects, materials, colliders and other game engine related subjects but above all I value the experience I got from all the meetings and working with fantastic people. I also programmed the game's login logic, which introduced me into hashing, salting, encoding and decoding. The project also depended on version control with GIT I was responsible for.

Unfortunately, the code is confidential, but some external information about the project is available here (Finnish).
